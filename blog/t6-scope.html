Scope

Explain to a non-tech friend:

  • What 'scope' is, and how it works in JavaScript:

    • Scope refers to the range of visibility of variables within code. In JavaScript, there are two kinds of scope; global and local scope.

    Global scope is anywhere that is not within the curly brace confines of functions and block statments. Local scope is the scope which exists only inside these functions and block statements; so, in curly braces { }.

    Variables declared within global scope are visible and accessible to any part of the code, while local scope/s and variables declared within them are visible and accessible only to code within that shared local scope, (and invisible to other parts of the code). There is only one Global Scope, while there may be infinite local scopes created through infinite declarations of functions and blocks, and functions and blocks within functions and blocks, etc.
