What is grid based design?

Grid based design refers to media that is designed in a grid, like most print media, and most websites. Grid based designs helps us read and navigate webpages easily.

Whats all the hype about responsive webpages?

Responsive webpages are adaptive to display on different devices, and responsive design means often changing grid layouts to be optimised for the use of different devices.

What is semantic structure?

Semantic structure is represented by semantic tags, e.g; p is used to create a paragraph, and h1, h2, h3 etc tags refer to header while numbers following are ordered from most important (1) to less important. Semantic structures are necessary to make code easily identifiable to humans.
