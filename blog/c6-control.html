When programming, what are some of your triggers? What causes a disproportionate emotional response?

I get most upset when I understand how the code should be put together (at least in basic terms), but do not know at all how to execute it independantly. Having examples of what the code should look like is really helpful, so at least I can try to reverse engineer it. There were less of these examples in sprint 6 so my disproportionate emotional responses were more frequent. In these situations, I get stuck re-reading the waffle ticket instructions over and over, which does help, but gives me no clues for how to actually do-the-thing.

How did you find the Siberian North Railroad exercise (Stop, Breathe, Notice, Reflect, Respond)?

The exercise which has you conjure up a past scenario and play out practicing the steps worked surprisingly well for me.

This chapter made me think more about how quickly I react to things that upset me, which most of the time is in either extreme states of fight-y or flight-y. This exercise has been the most helpful so far, so I will try it again next time I experience some conflict.
