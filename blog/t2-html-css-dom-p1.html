What is meant by boxifying design?

Boxifying design refers to dividing elements of a webpage's content into a box—headers, paragraphs, images, and hyperlinks are all boxes within the larger box of a webpage.

What is the box model?

The box model consists of 4 elements which make up the total area of an individual box. These are: Content — the content of the box itself/the element which the box exists for, Padding — the space which separates the content from the border, Border — the border around the box, and Margin — the space separating the border from other elements/boxes on the webpage.
