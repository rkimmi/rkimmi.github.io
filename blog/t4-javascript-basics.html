How does JavaScript compare to HTML and CSS?

HTML defines the basic skeletal structure of a webpage and contains its basic essential content. CSS renders this structure and content to look either good, bad, neither, or both, et cetera. JavaScript is the language used to define and carry out more complex web functions — this being reason for interactive and user-resposive web content.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control Flow describes that code will be executed and/or evaluated in a perscribed order. Loops tell the code to run and/or repeat an action within time specified by given variables. if/else statements provide a good example of what this looks like actually, e.g; we can define that if the weather is nice, and it is the weekend: go to the beach! else, don't. This hypothetical code can additionally be set to loop only until Summer is over.

Explain the difference between accessing data from arrays and objects.

Arrays hold lists of items. Each item in the list is given an ordered value with the first item being 0, the second given the value of 1, and so on. To access this data, items can be called upon using their value.

An object can hold data, including arrays, as well as data functions. Accessing data from objects requires calling on data with either dot or bracket notations.

Explain what functions are and why they are useful.

Functions define operations or tasks designed for code to carry out when called upon. This gives JavaScript its ability of mobilising interactive, user responsive content. It also enables code to perform more complex behind-the-scenes processes. Functions tell data how to and when to perform tasks and/operations when a function is applied.
