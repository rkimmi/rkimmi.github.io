How does JavaScript compare to HTML and CSS?

HTML defines the basic skeletal structure of a webpage and contains its basic essential content. CSS renders this structure and content to look either good, bad, neither, or both, et cetera. JavaScript is the language used to define and carry out more complex web functions — these being functions that are interactive and user-resposive.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control Flow describes that code will be executed and/or evaluated in a perscribed order. Loops tell the code to run and/or repeat an action within time specified by given variables. if/else statements provide a good example of what this looks like actually, e.g; we can define that if the weather is nice, and it is the weekend: go to the beach! else, don't. This code can be set to loop only until Summer is over.

Explain the difference between accessing data from arrays and objects.

An array holds lists, with each defined item in the list given a value with the first as 0, the second as 1, and so on. To access this data, it can be called upon using the value of items. An object can hold data, including arrays, as well as data functions. Accessing data from objects requires calling on data with either dot or bracket notations.

Explain what functions are and why they are useful.

Functions define operations or tasks designed for the code to carry out when called upon. This gives JavaScript its point of interactive, user responsive content but also enables code to perform behind-the-scenes processes. Functions tell data to how and when to perform these tasks and/operations when the function is applied.
